# 💧 AquaTrack – Smart Hydration Reminder App

## 📘 Building a Real-World Mobile Application

I created AquaTrack to solve a simple but common problem — forgetting to drink enough water during the day. This project focuses on building a practical mobile application that helps users track hydration while adjusting water intake recommendations based on real-time weather conditions and personal activity levels.

The goal of this project was to strengthen my understanding of mobile application architecture, API integration, and persistent data handling while developing a useful real-world tool.

---

## 🎯 Development Journey

### **Timeline & Approach**
- **Duration**: Personal project developed iteratively over multiple development sessions  
- **Process**: Building features step-by-step and testing them on a real device  
- **Focus**: Mobile UI development, API integration, and notification systems  

---

## 🧠 Skills I Developed Through This Project

### **Mobile App Development**
- Building multi-screen applications using **React Native**
- Implementing navigation between screens using **React Navigation**
- Creating reusable components for maintainable code
- Testing and debugging the application on a real device using **Expo Go**

### **API Integration**
- Fetching real-time weather data from the **Open-Meteo API**
- Using device GPS location to personalize weather information
- Handling loading states and refresh logic for API calls
- Processing external data to influence hydration recommendations

### **Data Persistence**
- Storing hydration data using **AsyncStorage**
- Saving user profile information and preferences
- Maintaining hydration history and streak tracking
- Implementing daily reset logic for hydration tracking

### **Notification Systems**
- Scheduling local hydration reminder notifications
- Designing reminder logic based on user behavior
- Implementing quiet hours to avoid late-night notifications
- Managing notification timing and cancellation logic

### **Feature Engineering**
- Creating personalized hydration goals based on user profile data
- Adjusting water intake recommendations using temperature and humidity
- Tracking hydration progress throughout the day
- Implementing streak tracking and achievement-style motivation
